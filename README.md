# chenle02 community defaults

This public repository provides account-wide community health defaults for
open-source repositories owned by [chenle02](https://github.com/chenle02).

- `.github/FUNDING.yml` enables the default Sponsor button and links to
  [GitHub Sponsors](https://github.com/sponsors/chenle02).
- An individual repository can override a default by providing its own file of
  the same type.
